# 📒 Contact Book (Python Tkinter)

A simple and user-friendly **Contact Book desktop application built using Python and Tkinter**.  
This application allows users to store, manage, search, and update contact information through a graphical user interface.

The project demonstrates **basic Python GUI development, data management, and user interaction using Tkinter widgets**. It is designed as a beginner-friendly project for learning how to build practical desktop applications with Python.

---

##  Features

-  Add new contacts
-  Search contacts by name or phone number
-  View selected contact details
-  Update existing contact information
-  Delete contacts
-  Clear input fields
-  Display contacts in a list
-  Lavender-themed graphical user interface

## 🎮 How to Use

1. Enter contact details such as **Name, Phone, Email, and Address**.
2. Click **Add** to save the contact.
3. Select a contact from the list and click **View Selected Contact** to see its details.
4. Modify the fields and click **Update** to update the contact.
5. Click **Delete** to remove a contact.
6. Use the **Search** bar to find contacts by name or phone number.
7. Click **Clear** to reset the input fields.
